Abstract

The World Happiness Report annually ranks over 150 countries. This project aims to visualize happiness scores based on several factors. The relationship between features that affect happiness will be identified and explored through statistical analysis to determine the correlation between the various features and happiness score. This will be displayed in a dashboard format depicting the map of the world and will have the ability to toggle the factors measuring quality of life and the happiness index upon zooming into different countries or regions.

Visualization explanation

The main view displays the happiness index for each country in the world based on the scale of the color map used. One quantitative feature can be shown on the map at a time. Upon hovering over a country on the map, the user will be presented with pop-up information about the country that is not intended to be displayed in a visual format, including the numeric value of the happiness score of that specific country.

Additionally, upon scrolling down from the world map view, a scatterplot matrix displays the relationships between happiness index and other quantitative attributes, such as the life expectancy or perceived corruption score, grouped visually by color into different geographical regions. The regional distinction also allows for an easier view of emerging patterns, especially with the available brushing and linking to highlight specific regions.
